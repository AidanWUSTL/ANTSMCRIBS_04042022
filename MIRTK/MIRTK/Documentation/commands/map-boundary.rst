Arguments
---------

.. option:: domain

   Tesselation of the surface of the input shape.

.. option:: codomain

   File name of piecewise linear complex or name of primitive shape:
   
   - disk (default)
   - square
   - polygon

.. option:: output

   Output point set with boundary values as point data.

---------------

.. option:: -radius <float>

   Radius of primitive codomain shape. When non-positive, the radius is

   Name of boundary values point data array. (default: Map)

.. option:: -mask <string>
